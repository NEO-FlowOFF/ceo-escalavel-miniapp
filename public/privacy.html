Política de Privacidade

Última atualização: Janeiro de 2026

1. Introdução

O Agente Flow (@AgenteFlow_Bot) é um Mini App do Telegram desenvolvido pela NEØFLW. Esta política descreve como coletamos e usamos suas informações.

2. Dados Coletados

Quando você utiliza o Agente Flow, podemos acessar:

  • Nome de usuário (para personalização)
  • ID do Telegram (para salvar progresso)
  • Dados de progresso do jogo

Não coletamos: Número de telefone, contatos, localização ou dados sensíveis.

3. Uso dos Dados

Os dados são usados exclusivamente para salvar seu progresso, personalizar a experiência e melhorar o app.

4. Armazenamento

Dados são armazenados localmente no seu dispositivo (LocalStorage) ou no Telegram Cloud Storage. Não utilizamos servidores externos.

5. Compartilhamento

Seus dados não são vendidos ou compartilhados com terceiros.

6. Seus Direitos

Você pode apagar dados limpando o cache do navegador ou solicitar informações via @neomello.

7. Contato
